# SeerrAPN

SeerrAPN is an [APN](https://d.lumaa.fr/AosqMi) service for [Seerr](https://github.com/seerr-team/seerr).

## Events supported

- New Request
- Request Denied
- Movie/Show Available

## Filters

You can make it so that users have only specific notifications:

- None: `0`
- New Request: `1`
- Request Denied: `2`
- Movie/Show Available: `4`
